//See Readme.md for conventions used in this script. 
//The following 3 events occur in a placeholder.
//Initially the dog called Nida is sitting.
//Nida starts running.
//Sometime later Nida starts barking.

//While this is happening an observer called Me sees something black running.
//The observer realises that it is a running dog.
//The observer realises that the running dog is Nida.
//The observer notices Nida barking.

//The observer has parent object animal.
//The child objects human and dog both have the parent animal.
//My_Nida is the memory Me has of Nida. My_Nida has parent object dog.

/////////////////////////////////////////////////////////////////////////////////////////////////////
//Several commands:
DisplaySOs *	//display all the SOs with SQs.
//RandomWalk 30
//GenerateScript *

/////////////////////////////////////////////////////////////////////////////////////////////////////
//initialise the objects.
INIT Includes SO_EVENT
INIT Includes SO
SO Includes RO

SO_EVENT has property time
SO_EVENT includes RO_EVENT
An RO_EVENT has property place. 
An RO_EVENT has property weather. 
An RO_EVENT has property instance. //implies instantaneous event.
INIT is a type of RO_EVENT.
INIT Includes Placeholder
INIT Includes Nida.
INIT Includes Me.
INIT Includes SO_timeline
SO_timeline Includes RO_timeline
Me Includes RO_Me

Nida is a type of SO

Me is a type of SO
RO_Me is a type of RO
Nida Includes RO_Nida.
RO_Nida is a type of RO

//And use inheritance to indicate that My_Nida is derived from the dog.
//Define my idea of a generic dog.
Me includes animal
Me includes human
human is a type of animal
Me is a type of human
Me includes dog
dog has the property FrontLeftLeg 
//etc
//dog has the property 	BackLeftLeg
//dog has the property 	FrontRightLeg
//dog has the property 	BackRightLeg
//Add my SO of Nida within My generic dog.
dog is type of animal
Me includes My_Nida
My_Nida is type of dog
My_Nida has property Can Sit = true

/////////////////////////////////////////////////////////////////////////////////////////////////////
MeetNida includes Nida
MeetNida includes Me

//Set up the initial steady state with Nida Sitting.
MeetNida is a type of SO_timeline. 
The MeetNida has the property place and it is Bakkeveen. 
The MeetNida has the property weather and it is wet.
Nida include_right NidaInBakkeveen
NidaInBakkeveen is a type of Placeholder
NidaInBakkeveen includes _MeInBakkeveen
NidaInBakkeveen include_right NIDA_RUNS
/////////////////////////////////////////////////////////////////////////////////////////////////////
//Nida starts running.
//The time it takes for Nida to start running could be broken up into several events but to keep things simple 
//NIDA_RUNS is an instantaneous event associated with Nida starting to run and therefore no longer sitting.
//The relevant point is that the observer Me does not register that Nida is running.
//And for this reason Me is not included in the event.

NIDA_RUNS includes RO_NIDA_RUNS
NIDA_RUNS is a type of SO_EVENT
RO_NIDA_RUNS is a type of RO_EVENT
NIDA_RUNS has property time = 2026-01-01 12:35:00.000.
RO_NIDA_RUNS includes INPUT1
INPUT1 is a type of Placeholder
//The INPUT1 includes Nida.
RO_NIDA_RUNS includes OUTPUT1
OUTPUT1 is a type of Placeholder
NidaInBakkeveen has property Sitting = false
NidaInBakkeveen has property  Running = true

NidaInBakkeveen include_right Nida_running
//We enter a steady state in which Nida is running but the observer has not realised that yet.
Nida_running is a type of SO_timeline. 
//Nida_running includes Nida
ENDOBJECT NIDA_RUNS

Me include_right MeInBakkeveen
MeInBakkeveen is a type of Placeholder

/////////////////////////////////////////////////////////////////////////////////////////////////////
//Set up the initial steady state with Nida Sitting.
//ObserverMeetNida is a type of SO_timeline. 
The MeInBakkeveen has the property place and it is Bakkeveen. 
The MeInBakkeveen has the property weather and it is wet.

MeInBakkeveen include_right SEE_SOMETHING
SEE_SOMETHING is a type of SO_EVENT
SEE_SOMETHING has property time = 2026-01-01 12:35:00.000.
SEE_SOMETHING includes RO_SEE_SOMETHING
RO_SEE_SOMETHING is a type of RO_EVENT
RO_SEE_SOMETHING includes INPUTa
INPUTa is a type of Placeholder

RO_SEE_SOMETHING includes OUTPUTa
OUTPUTa is a type of Placeholder

MeInBakkeveen include_right something_running

ENDOBJECT SEE_SOMETHING

MeInBakkeveen include_right SEE_RUNNING_DOG
SEE_RUNNING_DOG is a type of SO_EVENT
SEE_RUNNING_DOG has property time = 2026-01-01 12:35:00.000.
SEE_RUNNING_DOG includes RO_SEE_RUNNING_DOG
RO_SEE_RUNNING_DOG is a type of RO_EVENT
RO_SEE_RUNNING_DOG includes INPUTb

RO_SEE_RUNNING_DOG includes OUTPUTb

ENDOBJECT something_running

/////////////////////////////////////////////////////////////////////////////////////////////////////
//The observer realises that Nida is running.

//If we didn't know it already we conclude that My_Nida can Run.
My_Nida has property Can Run = true
ENDOBJECT dog_running

/////////////////////////////////////////////////////////////////////////////////////////////////////

//A steady state is reached with Nida running and the observer is also aware of that.

ENDOBJECT SEE_NIDA_RUN

/////////////////////////////////////////////////////////////////////////////////////////////////////
